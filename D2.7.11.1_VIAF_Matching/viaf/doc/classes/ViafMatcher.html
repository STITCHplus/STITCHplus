Public Class methods

scores(labels, birth_date, death_date, verbose = true, delay = 3)

Takes an array of strings for labels, optionally year of birth (int) and year of death (int) Returns the scores for a set of labels as an array of hashes in the form: 

 - h[:score] (int): the score of the match
 - h[:label] (string): the VIAF label for the person (preferred, or alternative)
 - h[:uri] (string): the VIAF linked data URI for the person's metadata
